## ğŸ“ Common Tasks

### Creating Your First Task

1. Go to **Tasks** tab
2. Tap **+ icon** (top right)
3. Enter task details:
   - Title: "Finish project proposal"
   - Description: Optional details
   - Priority: Select High
   - Due Date: Toggle on, select tomorrow
4. Tap **Add**
5. Task appears in list with orange badge (High priority)

### Starting Your First Habit

1. Go to **Habits** tab
2. Tap **+ icon**
3. Choose habit details:
   - Name: "Morning Exercise"
   - Description: "30 min workout"
   - Icon: Select dumbbell
   - Color: Pick orange
   - Goal: Daily goal = 1
4. Tap **Create**
5. Tap **Mark Done** to complete today
6. Watch progress ring animate!

### Changing Theme

1. Go to **Settings** tab
2. Tap **Theme** row
3. Browse 6 available themes
4. Tap theme card to select
5. Watch colors smoothly transition
6. Tap **Done**

### Enabling Dark Mode

1. Go to **Settings** tab
2. Find **Dark Mode** toggle
3. Switch on/off
4. Entire app adapts instantly

## ğŸ› Troubleshooting

### App Won't Build
- Clean build folder: `Cmd + Shift + K`
- Close and reopen Xcode
- Update to latest Xcode version

### Notifications Not Working
- Check simulator/device settings
- Ensure permission was granted
- Restart app if needed

### Theme Not Changing
- Wait for animation to complete
- Check Settings > Appearance
- Restart app if theme doesn't persist

### Data Not Saving
- Check file permissions
- Clear UserDefaults (advanced)
- Reinstall app
